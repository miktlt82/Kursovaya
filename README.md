# RAMMPROJECT SHOP

Интернет-магазин мерча трибьют-шоу **RAMMPROJECT** (Rammstein tribute show).
Курсовой проект: каталог товаров, корзина, оформление заказа. Фронтенд на чистом
HTML/CSS/JavaScript, бэкенд на **Node.js + Express**, база данных — **SQLite**.

## Возможности

- Каталог товаров с фильтрами по категориям (футболки / майки / тур-мерч).
- Карточка товара: фото, описание, выбор размера, таблица размеров.
- Корзина: добавление, изменение количества, удаление; сохраняется в `localStorage`.
- Оформление заказа с проверкой полей формы — заказ записывается в базу данных.
- Адаптивная вёрстка: меню-бургер, две колонки на телефоне, «липкие» фильтры, свайп-закрытие корзины.
- Анимированный фон на Canvas.

## Запуск

Нужен установленный [Node.js](https://nodejs.org/) (версия 18+).

```bash
npm install      # установить зависимости (один раз)
npm start        # запустить сервер
```

Затем открыть в браузере: **http://localhost:3000**

База данных `shop.db` создаётся автоматически при первом запуске и наполняется
каталогом из встроенного seed (см. `server.js`).

```
├── index.html        # разметка страницы
```

## REST API

| Метод | Путь | Назначение |
|---|---|---|
| `GET` | `/api/products` | список всех товаров |
| `GET` | `/api/products/:slug` | один товар по идентификатору |
| `POST` | `/api/orders` | оформить заказ (тело: `{ customer, items }`) |
| `GET` | `/api/orders` | список заказов с позициями |

## Структура базы данных

- **products** — каталог: `slug`, `category`, `name`, `price`, `image`, `description`, `sizes`, `sold_out`, `size_chart`, `tag`.
- **orders** — заказы: `order_no`, данные покупателя, `total`, `created_at`.
- **order_items** — позиции заказа: ссылки на `orders` и `products`, `size`, `qty`, `price`.
